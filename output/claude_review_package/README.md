# AquaGuide Claude Review Package

这个目录是给 Claude 审阅项目用的精简文档包。

推荐使用方式：

1. 先把 `Claude_Review_Prompt.md` 的内容复制给 Claude。
2. 再上传或粘贴 `AquaGuide_Claude_Review_Packet.md`。
3. 如果 Claude 需要看关键文件位置，再给它 `File_Index_For_Claude.md`。

文件说明：

- `AquaGuide_Claude_Review_Packet.md`：主审阅文档，包含产品、技术、数据、模块、风险和审阅要求。
- `Claude_Review_Prompt.md`：可以直接复制给 Claude 的提示词。
- `File_Index_For_Claude.md`：关键文件索引。

注意：这个包没有包含 `.env.local`、API Key、完整源码和图片大文件。
