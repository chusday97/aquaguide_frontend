# AquaGuide 交互体验重构 PRD

## 目标

让水族新手用更短的路径完成查看收藏、巡检、混养判断与 AI 建缸规划；所有可点击元素都必须产生明确结果。

## 本轮范围

- 新增“我的水族册”，合并种草、养护收藏、生命纪念与成就勋章。
- 将用户界面的 Copilot 统一改称“AI 建缸助手”。
- 按内容、详情、任务、确认四类重构交互表面。
- 删除重复、无结果或无法预判行为的按钮。

## 成功标准

- 从全局导航一次进入水族册，一次打开内容详情。
- 每日检查一次开始、结果一次进入补救文章。
- 图鉴选择两种生物后直接出现 Mini 结论。
- 所有写操作有结果反馈，所有破坏性操作有二次确认。
- 8 枚勋章可根据旧数据追溯并实时计算。

## 不在本轮

- 积分、等级、排行、勋章分享。
- 数据库或 Supabase 迁移。
- 3D 与物种大代码块拆包。
