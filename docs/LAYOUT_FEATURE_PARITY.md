# AquaGuide 设备布局与功能对照

## 判定原则

- `phone`：真实手机，使用 `MobileAppShell`。
- `desktop`：桌面浏览器、平板和桌面触控设备，使用 `DesktopAppShell`。
- 启动时只判定一次；缩放或调整浏览器宽度不会切换布局。
- 两个应用壳共享路由、数据服务和业务动作，布局组件不得自行保存业务数据。

## 核心页面功能对照

| 页面 | 共享数据与动作 | 手机布局入口 | 桌面布局入口 |
| --- | --- | --- | --- |
| 我的鱼缸 | 鱼缸资料、缸内生物、设备、诊断记录 | 底部导航“我的鱼缸” | 左侧栏“我的鱼缸” |
| 图鉴 | 物种库、收藏服务、统一混养规则引擎 | 底部导航“图鉴” | 左侧栏“图鉴” |
| 养护百科 | 文章库、收藏服务、问题定位 | 底部导航“养护百科” | 左侧栏“养护百科” |
| 种草图鉴 | 现有物种收藏 key 与收藏服务 | 图鉴页收藏入口 | 左侧栏“种草图鉴” |
| 养护收藏 | 现有养护收藏 key 与收藏服务 | 养护页收藏入口 | 左侧栏“养护收藏” |
| Mini 混养 | `species_only` 规则范围、已选物种 | 图鉴选择栏 | 图鉴选择栏 |
| 完整混养 | `tank` 规则范围、当前鱼缸环境 | 图鉴完整计算 | 图鉴完整计算 |
| 每日检查 | 当前鱼缸快照、诊断规则、当日记录 | 鱼缸核心操作区 | 鱼缸核心操作区 |

## 变更门禁

新增页面功能时必须同时核对手机和桌面入口。若两端表现不同，只允许视图结构不同；收藏、混养、诊断及鱼缸写入必须调用同一共享服务。自动验收至少覆盖桌面窄窗仍为桌面壳、iPhone 为手机壳、iPad 为桌面壳。
