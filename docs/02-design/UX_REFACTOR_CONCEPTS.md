# AquaGuide 交互与视觉方案

## 方案 A：自然水族册（实施方案）

- 保留 AquaGuide 现有米白、苔绿和水体蓝，使用纸册式摘要区与柔和玻璃卡片。
- 桌面详情从右侧滑入，手机详情从底部进入；长任务进入沉浸式单焦点流程。
- 勋章采用水滴形徽章与低饱和金属层次，锁定状态仍显示进度和下一步。

## 方案 B：仪表工作台

- 强调数据密度和快捷动作，适合资深用户。
- 缺点是会放大当前信息过载问题，因此不采用。

## 方案 C：沉浸水下卡片

- 使用更强的渐变、漂浮和景深效果。
- 视觉表现突出，但会与“路径简短、重点明确”冲突，因此只保留轻量动效语言。

## 关键版式

- 水族册桌面：顶部摘要 + 横向页签 + 多列卡片。
- 水族册手机：紧凑摘要 + 吸顶页签 + 单列卡片。
- 详情首屏：标题、一个结论、最多三个关键点、一个主操作。
- 转场时长：160–220ms；不设置纯装饰按钮或不可操作视觉控件。
